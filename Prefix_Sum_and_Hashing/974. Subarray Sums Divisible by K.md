In given problem's array may contain -ve numbers also so the best method to solve is **prefix-sum+HashMap**

# Approach
<!-- Describe your approach to solving the problem. -->
the approach is slightly different to the approach of the *sum of the subarrays count whose sum=k*
**but initially we need *mp[0]=1* why because to handle the edge cases where the remainder is exactly zero**
1. iterate the loop
2. in the loop get the prefix sum 
3. here instead of storing the sum we store the remainder of the sum i.e. *rem=sum % k* 
4. then we will check it in the hashMap whether that remainder is already present or not 
5. if yes then we will add that number to the count
6. later we will increment the value of that hashmap *(mp[rem]++)*
7. finally we return the count.

# Code
```cpp []
class Solution {
public:
    int subarraysDivByK(vector<int>& nums, int k) {
        unordered_map<int, int> mp;
        int sum = 0;
        int count = 0, rem = 0;
        mp[0] = 1;
        for (auto num : nums) {
            sum += num;
            rem = sum % k;
            if (rem < 0)
                rem += k;
            if (mp.find(rem) != mp.end()) {
                count += mp[rem];
            }
            mp[rem]++;
        }
        return count;
    }
};
```
