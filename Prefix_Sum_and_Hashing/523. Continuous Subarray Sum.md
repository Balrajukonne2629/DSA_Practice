upon seeing the problem we can see the wording of the multiple of k so from this we can understand that it can be solved using **prefix-sum+modulo+Hashmap**

# Approach
<!-- Describe your approach to solving the problem. -->
the approach is 
1. first we create a hashmap and initialize it with the mp[0]=-1 
> we initialize it -1 because to not miss the starting value of the array which is at index=0.
2. intiate the loop along with prefix sum
3. store the remainder of the prefix sum and the given value
4. check whether the rem is already exists in the map
    >if yes then store the existing rem index in r
    then if the **i-r>=2** then return true
5. else store the rem in the map along with it's index.

# Code
```cpp []
class Solution {
public:
    bool checkSubarraySum(vector<int>& nums, int k) {
        int sum = 0, r = 0, l = 0;
        unordered_map<int, int> mp;
        mp[0] = -1;
        for (int i = 0; i < nums.size(); i++) {
            sum += nums[i];
            int rem = sum % k;
            if (mp.find(rem) != mp.end()) {
                r = mp[rem];
                if (i - r >= 2) {
                    return true;
                }
            } else {
                mp[rem] = i;
            }
        }
        return false;
    }
};
```
