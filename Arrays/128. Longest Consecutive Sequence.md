# Approach
<!-- Describe your approach to solving the problem. -->
the approach i followed is 
1. first store all the elements of the array to a set
2. then iterate a loop for all the elements in the set
3. in that loop itself check the condition to find the starting of the sequence i.e *st.find(num-1)==st.end()*
4. if the condition is satisified then iterate a while loop to find the next elements of the sequence from the set where the while loop condition is *st.find(num+j)!=st.end()* in the while loop increment the count and j.
5. After the condition assgin the max value b/w count and maxCount to the maxCount.
6. return the maxCount

# Code
```cpp []
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_set<int>st;
        int count=0,maxCount=0;
        for(int i=0;i<nums.size();i++)
        {
            st.insert(nums[i]);
        }
        for(int num:st)
        {
            if(st.find(num-1)==st.end())
            {
                count=1;
                int j=1;
                while(st.find(num+j)!=st.end())
                {
                    count++;
                    j++;
                }
            }
            maxCount=max(maxCount,count);
            count=0;
           
        }
        return maxCount;
        
    }
};
```
