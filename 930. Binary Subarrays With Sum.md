This problem seems to be solved by the sliding window pattern with a small change in logic

# Approach
<!-- Describe your approach to solving the problem. -->
the approach is same as like to the problem of solving the sum of subarrays whose sum is <=k but we need exactly the number of subArrays whose sum=k so just call the same functions twice with target as k and k-1 then return the difference between them which is the total number of subArrays whose sum is exactly k.

# Code
```cpp []
class Solution {
public:
    int numSubarraysCount(vector<int>& nums, int goal) {
        int sum = 0, l = 0, count = 0;
        if (goal < 0)
            return 0;
        for (int r = 0; r < nums.size(); r++) {
            sum += nums[r];
            while (sum > goal) {
                sum = sum - nums[l];
                l++;
            }

            count += (r - l + 1);
        }
        return count;
    }
    int numSubarraysWithSum(vector<int>& nums, int goal) {
        return numSubarraysCount(nums, goal) -
               numSubarraysCount(nums, goal - 1);
    }
};
```
