# Approach
<!-- Describe your approach to solving the problem. -->
The approach is the mix of hashset and variable sliding window.
1. first create a hashset of char
2. initiate a for loop for iterating through the string
3. initiate the while loop with condition **while(mp.find(char)!=mp.end())** in it remove the left char of the window and increment the left.
4. then insert the char into the window *i.e. set*
5. assign the max length to the maxLen variable
6. return the maxLen

# Code
```cpp []
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_set<char> mp;
        int l = 0, maxLen = 0;
        for (int right = 0; right < s.size(); right++) {
            while (mp.find(s[right]) != mp.end()) {
                mp.erase(s[l]);
                l++;
            }
            mp.insert(s[right]);
            maxLen = max(maxLen, (right - l) + 1);
        }

        return maxLen;
    }
};
```
