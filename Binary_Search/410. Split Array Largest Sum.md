# Approach
<!-- Describe your approach to solving the problem. -->
The approach
-> we need a binary search for minimum sum and a isValid function to split the array
first we define the *isValid function*
1. we take array,k,and mid as parameters to this function
2. we will iterate a loop then we will add the values of the array 
3. while adding we check the condition whether the sum is <= mid
4. if yes then add the value to the sum 
5. if no then assign the array value to the sum and increment the count.
6. return true if the count is <=k else false.
**Now the Binary Search**
- here low=max value of the array,high is the sum of all values of the array
- iterate while loop with condition *low<high* (why this one because here we are not searching for a value so no low <= high)
- mid=low+(high-low)/2;
- if (isValid is true)=>then high=mid else low=mid+1
- return low

# Code
```cpp []
class Solution {
public:
    int splitArray(vector<int>& nums, int k) {
        int low = 0, high = 0;
        for (auto i : nums) {
            high += i;
            low = max (i, low);
        }
        while (low < high) {
            int mid = low + (high - low) / 2;
            if(isValid(nums,k,mid))
            {
                high=mid;
            }
            else
            {
                low=mid+1;
            }
        }
        return low;
    }
    bool isValid(vector<int>& nums, int k, int mid) {
        int sum = 0, count = 1;
        for (int i = 0; i < nums.size(); i++) {
            if (sum + nums[i] <= mid) {
                sum += nums[i];
            } else {
                sum = nums[i];
                count++;
            }
        }
        return count <= k;
    }
};
```
