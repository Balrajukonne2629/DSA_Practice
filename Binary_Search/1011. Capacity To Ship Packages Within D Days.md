# Intuition
<!-- Describe your first thoughts on how to solve this problem. -->
Here we need to ship all the packages within D days such that the capacity of the ship is minimum.
Each day we can take contiguous packages and the total weight in a day should not exceed the capacity.

So instead of trying all possibilities, we assume a capacity and check whether it is possible to ship all packages within D days.

> 👉 This is similar to Allocate Books / Split Array, so we use Binary Search on Answer

# Approach
<!-- Describe your approach to solving the problem. -->
The approach → we need a binary search for minimum capacity and an isValid function to check feasibility.

🔹 isValid Function

We take weights, days, and mid (capacity) as parameters.

Initialize:
`sum = 0` (current day load)
`count = 1` (at least one day)
Iterate through the array:
If `sum + weights[i] <= mid` → add to current day
Else:
Increment day count
Assign weights[i] to new day `(sum = weights[i])`
Finally:
Return true if `count <= days`
Else false
🔹 Binary Search
- Define:
`low = max(weights) `→ minimum possible capacity
`high = sum(weights)` → maximum possible capacity
- Iterate:
```
while (low < high)
```
- Find mid:
```
mid = low + (high - low) / 2;
```
- Check:
````
If isValid(mid) → high = mid
Else → low = mid + 1
````
- Return:
```
return low;
```

- Time complexity:

- Space complexity:

$$O(Nlog(sum(Weights)))$$
# Code
```cpp []
class Solution {
public:
    int shipWithinDays(vector<int>& weights, int days) {
        int low = 0, high = 0;
        for (int i = 0; i < weights.size(); i++) {
            low = max(low, weights[i]);
            high += weights[i];
        }
        while (low < high) {
            int mid = low + (high - low) / 2;
            if (isValid(weights, days, mid))
                high = mid;
            else
                low = mid + 1;
        }
        return low;
    }

    bool isValid(vector<int>& weights, int days, int mid) {
        int sum = 0, count = 1;
        for (int i = 0; i < weights.size(); i++) {
            if (sum + weights[i] > mid) {
                count++;
                sum = weights[i];
            } else {
                sum += weights[i];
            }
        }
        return count <= days;
    }
};
```
