# Intuition
<!-- Describe your first thoughts on how to solve this problem. -->
We need the smallest subarray whose sum is at least the target. Since all numbers are positive, expanding the window increases the sum, and shrinking decreases it. This naturally suggests a sliding window approach
# Approach
<!-- Describe your approach to solving the problem. -->
firstly as it was a variable sliding window problem
1.  take a total sum variable and the minLength varable with INT_MAX as value assinged to it.
2. start a for loop with the iteration from the starting of the array to end with variable i or right and add the array element to the total sum variable.
3. in the for loop intialize a while loop with condition **sum>=target**
4. in the while loop assign the minimum value to the minLength from (minLength,i-l+1)and subtract the value of *nums[l] from totalsum*
5. return the minLength if "minLength!=INT_MAX" else return 0.

# Code
```cpp []
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) {
        int sum = 0, len = INT_MAX;
        int l = 0;
        for (int i = 0; i < nums.size(); i++) {
            sum += nums[i];
            while (sum >= target) {
                len = min(len, i - l + 1);
                sum -= nums[l];
                l++;
            }
        }
        if(len==INT_MAX)return 0;
        return len;
    }
};
```
